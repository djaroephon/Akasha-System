**Akasha System** adalah dashboard modern, estetik, dan kaya fitur yang dirancang khusus untuk pemain Genshin Impact. Dibangun menggunakan **Nuxt 3** dan **Tailwind CSS**, aplikasi ini berfungsi sebagai terminal pribadi untuk melihat data karakter, mengelola koleksi "Waifu", dan merencanakan tabungan Gacha kamu dengan Wishes Planner yang canggih.
